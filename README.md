# স্কুল ও মাদ্রাসা ম্যানেজমেন্ট সিস্টেম (School-Madrasa Management)

এটি একটি আধুনিক এবং রেসপনসিভ ওয়েব অ্যাপ্লিকেশন যা স্কুল বা মাদ্রাসার তথ্য প্রদর্শনের জন্য তৈরি করা হয়েছে। এতে অভিভাবকদের মতামত, এবং প্রাতিষ্ঠানিক তথ্য সুন্দরভাবে উপস্থাপন করা হয়েছে।

## Live Demo
[Live Link](https://school-madrasa-management.netlify.app/)


##  Technologies
- **Frontend:** React.js, JavaScript
- **Styling:** Tailwind CSS, Lucide React (Icons), React Icons
- **Animation:** Framer Motion
- **Data Fetching:** Axios
- **Deployment:** Netlify
